# Projeto Rede Daora - Aula 02
**Fundamentos Web para Iniciantes: Lógica, Estrutura e Estilo**

Bem-vindo(a) ao repositório da **Aula 02**! 🚀

Este projeto contém os slides apresentados em aula e materiais de apoio práticos abordando conceitos fundamentais de desenvolvimento web, incluindo Algoritmos a estrutura da Internet (Nuvem), o DOM e CSS.

## 📂 Estrutura do Projeto

Todo o conteúdo principal está detalhado na pasta `src/`:

* `src/slides.html`: Apresentação completa e customizada do curso em formato HTML.
* `src/apoio/`: Guias rápidos e explicações práticas sobre os conteúdos específicos vistos em aula.
* `src/index.md` / `src/links.md`: Material complementar e coleções de links úteis.

## 🛠️ Como acessar o material

Para visualizar os arquivos da aula em seu computador, você pode utilizar duas formas:

### Opção 1: Rodar o projeto localmente (Recomendado)

Este repositório utiliza o gerador de sites Eleventy. Se você já possui o [Node.js](https://nodejs.org/pt-br/) e um gerenciador de pacotes instalado (NPM ou PNPM), é possível rodar tudo facilmente:

1. Abra o terminal na raiz da pasta do repositório.
2. Instale as dependências executando:
   ```bash
   npm install
   ```
   *(Ou `pnpm install`, caso utilize o pnpm)*
3. Inicie o servidor local:
   ```bash
   npm start
   ```
4. Após o comando, acesse o endereço retornado no terminal (por padrão: `http://localhost:8080`) no seu navegador. O site inteiro estará funcionando!

### Opção 2: Acesso rápido aos arquivos

Caso queira checar o material rápido, vá na pasta baixada no seu computador e dê um clique-duplo nos arquivos de final `.html` (por ex: `src/slides.html` ou os arquivos dentro da pasta `src/apoio/`). O conteúdo já tem visuais organizados e você pode estudar por lá mesmo.

## 🎯 Assuntos Abordados na Aula

- **O Invisível:** Como a internet funciona, as requisições Cliente/Servidor, softwares compilados x interpretados e Algoritmos.
- **Construção e Marcação:** Criação de estrutura de página com HTML Semântico e entendimento da árvore familiar do navegador (DOM).
- **A Pintura e Estilo:** Lógica do CSS, entendendo a sintaxe de seletores, propriedades e valores, e a regra do *Box Model*.
